# ENSO Operations Dashboard

A complete GitHub Pages dashboard for tracking daily performance against the ENSO consultancy agreement targets.

## Features

- **Live Daily Reporting**: Track daily metrics in real-time
- **Interactive Dashboard**: Visual charts and progress tracking
- **Form Submission**: Complete daily report form with validation
- **Local Storage**: All data saved in your browser
- **Responsive Design**: Works on desktop and mobile
- **Export/Import**: Backup and restore your data
- **Performance Scoring**: Automatic calculation of daily performance
